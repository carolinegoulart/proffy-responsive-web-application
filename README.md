# :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/carolinegoulart/proffy-web-and-mobile-application.git
```
### üì¶  Running the API

```bash
# Go to server folder
$ cd Proffy/server

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
Access API at http://localhost:3333/

### üíª  Running the Web Project

```bash
# Go to web folder
$ cd Proffy/web

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
